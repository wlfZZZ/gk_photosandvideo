# GK Photos and Video

Premium aesthetic photography studio website built with Flask, HTML, CSS, and JavaScript.

## Tech stack

- Python
- Flask
- HTML templates with Jinja
- CSS
- Vanilla JavaScript

## Run locally

```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
python route.py
```

Open `http://127.0.0.1:5000` in your browser.
